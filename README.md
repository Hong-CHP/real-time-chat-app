# My Chat Project

My first full-stack real-time chat project, built with NestJS, Prisma, PostgreSQL, React, WebSocket, Docker, and Nginx!

## 🚀 Tech Stack

* **Backend:** NestJS, Prisma (ORM)
* **Database:** PostgreSQL
* **Real-time:** WebSockets (Socket.io)
* **Frontend:** React
* **DevOps:** Docker, Nginx
